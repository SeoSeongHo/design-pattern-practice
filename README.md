# 디자인 패턴 연습 프로젝트

## 개요
어느 순간부터 코드 구현을 위해, 구조를 생각할 때, 몇 가지 정해진 디자인 패턴만을 사용하기 시작했습니다.

물론, 상황에 잘맞는 디자인 패턴을 사용하여, 깔끔한 구조로 구현하려 노력하였지만, 
다양한 디자인 패턴을 알고 있는 것은, 더 나은 구조로 변경하는 데에 있어 도움이 될 것 같아 시작하게 되었습니다.

***
